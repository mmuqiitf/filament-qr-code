## [1.0.0] - 2026-01-07

### Added

- Initial release
- `QrCodeScannerInput` - Text input component with QR scanner modal
- `QrCodeScannerAction` - Standalone action for QR scanning
- `QrCodeScannerPage` - Pre-built page for QR scanning workflows
- `HasQrScanner` trait for custom page integration
- Support for Filament v3 and v4
- Support for Laravel 10, 11, and 12
- Configuration options for camera, scan mode, FPS, etc.
- Beep and vibration feedback on scan
- Manual input fallback
- Multi-language support (English, Indonesian)
- Dark mode support
